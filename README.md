# @xea/react-service

> 运行于ts-node环境，ts配置文件，基于webpack-chain，自定义再配置webpack的React快速开发项目脚手架

```json
{
 "scripts": {
    "serve": "xea-react-service serve",
    "build": "xea-react-service build"
  }
}

```

## Installation

```bash
npm install --save-dev @xea/react-service
```
